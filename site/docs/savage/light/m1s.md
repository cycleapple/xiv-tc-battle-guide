## 機制攻略

### 四連尖甲

四連尖甲是前2擊會對距離BOSS最近的人施放扇形範圍攻擊，第3\~4擊則會在第1\~2擊施放的相同位置再次發動。

![四連尖甲①](https://img.game8.jp/10298806/795582ba716f2eb848364895c4988ac2.png/show)

①TH先靠近BOSS十字散開，DPS在外圍X字稍微遠離BOSS。

![四連尖甲②](https://img.game8.jp/10298805/a8e532866081265e1c363689d3c86619.png/show)

②第1發扇形範圍發動。

![四連尖甲③](https://img.game8.jp/10298807/601509c51fec8d03ca51a464e72fd288.png/show)

③接著DPS靠近BOSS十字散開，TH遠離BOSS在X字位散開。

![四連尖甲④](https://img.game8.jp/10298804/bb8fda5b57ee750567eaaa3e0a452454.png/show)

④第2發範圍發動。

![四連尖甲⑤](https://img.game8.jp/10298803/295e67e9b2d8034e78af27014bc5af49.png/show)

⑤第3\~4發的範圍與第1、2發相同位置發動，在X字位等待即可。

採用犬丸式的解法。第1\~2擊按TH→DPS的順序靠近BOSS十字散開承接範圍。第3\~4擊會在第1\~2擊的十字範圍相同位置連續發動，全員在X字安全區等待即可。

### 貓生九命 + 二連尖甲 + 靈魂之影 + 雙重利爪或四爪

貓生九命是持續發出紅色氣場的機制，會召喚自身的分身。這個機制中會在二連尖甲後召喚2個分身。

二連尖甲是連續2次的半場攻擊，會先從舉起手臂的那一側開始攻擊。之後靈魂之影會命令分身執行BOSS本體剛才做過的行動（二連尖甲）。

2個分身的二連尖甲結束後，本體會詠唱雙重利爪或四爪其中之一。雙重利爪是以治療為目標的分攤攻擊，MT組和ST組分開承接；四爪則是配對分攤，按巨集的十字配對承接。

### 血腥抓撓

傷害非常高的全體物理屬性攻擊。除了使用減傷與護盾外，由於是物理屬性的全體攻擊，近戰職業務必使用「牽制」。

④接著坦克成為擊飛對象。近戰職業使用內側的全新格子。

捕鼠破壞的地板恢復後變成8格地板，其中4格是全新地板、4格是損壞的地板。分身會針對TH或DPS，在玩家所在位置產生十字範圍攻擊，連續4次。被選為目標的玩家要在全新地板上承接分身的攻擊。遠程職業使用較遠的2格，近戰職業使用較近的2格。模仿之貓的攻擊分為原地暈眩或擊飛兩種。

#### 分身攻擊的判斷方式

![右手發光](https://img.game8.jp/10299081/cabcc2fca3c5258dfd18daf0d0fff7cb.jpeg/show)

右手發光：移動到玩家背後用上鉤拳擊飛。會朝自己面向的方向飛出。

![左手發光](https://img.game8.jp/10299083/0c6bd4fb5bb5dcced3e07d8f9c4ee01b.jpeg/show)

左手發光：在玩家所在位置攻擊並原地暈眩。

#### 單格處理法

![單格處理法](https://img.game8.jp/10299073/a8a2fe30037e13919cc39f2c04308515.jpeg/show)

分身的擊飛可以在1格內完成處理。在1格的4個角落等待並面向對角，即使被擊飛也能落在同一格內。這個方法的優點是不需要判斷分身是暈眩還是擊飛，而且十字範圍的出現位置也更容易判斷。

### 貓跳二連尖甲 → 靈魂之影

與貓跳四連尖甲相同，BOSS維持面向不變朝線的方向滑行移動，施放二連尖甲的連續2次半場攻擊。在移動目的地的安全區等待，第1擊發動後衝進跡地迴避。之後的靈魂之影會對南北其中一個分身用線連接，保存貓跳的移動方向和二連尖甲的攻擊順序。分身之後會以相同移動方向滑行並施放二連尖甲，要記住順序。

![貓跳二連①](https://img.game8.jp/10299504/293d4045947b1a63e3b19711497278cd.png/show)

![貓跳二連②](https://img.game8.jp/10299515/2aa45660d11246bee475a2be7d99ad57.png/show)

②朝線的方向滑行移動，同時從發光的手開始半場攻擊。

![貓跳二連③](https://img.game8.jp/10299516/e4c29e1686a1f0ad2336f50d6e68506a.png/show)

### 貓跳四連尖甲 → 靈魂之影

從貓跳二連尖甲移動到的位置施放貓跳四連尖甲。處理方式與前半的四連尖甲相同，但這裡是以BOSS面向為基準散開。因此要事前確認BOSS的面向和自己的散開位置。之後同樣會用靈魂之影保存移動方向和攻擊給分身。BOSS往左還是往右移動，之後會很重要，記住方向。

![貓跳四連B①](https://img.game8.jp/10299523/763bd4dd3d43a51a5e0a8f2d99a0f5dc.png/show)

### 暴風裂 + 二連尖甲（分身）

保存了二連尖甲的分身與BOSS用線連接後，BOSS施放暴風裂，分身施放二連尖甲。分身會朝與之前貓跳二連尖甲相同的方向移動，以相同順序施放左右半場攻擊，回想保存的順序處理。暴風裂是以治療2人為目標的直線分攤攻擊，以BOSS面向為基準正面MT組、背面ST組集合分攤。暴風裂和二連尖甲第1擊同時著彈，因此BOSS正面背面要稍微往安全側偏移。

![暴風裂①](https://img.game8.jp/10299641/f360029dbc623bf91300dd93a0fd675c.png/show)

①保存了二連尖甲的分身與BOSS用線連接。MT將BOSS引導到分身移動目標的數字標記位置。

![暴風裂②](https://img.game8.jp/10299643/606b85931cbd0030a4bd35e1b838a497.png/show)

②分身移動後，二連尖甲第1擊與暴風裂同時著彈。以BOSS面向為基準，正面MT組、背面ST組分開。圖示為BOSS面向南方的範例。

### 捕鼠 → 模仿之貓 + 超暗影或碎裂尖甲

與前半相同的捕鼠後，後半的模仿之貓會在分身的攻擊同時施放超暗影（直線分攤）或碎裂尖甲（3人扇形範圍）。看BOSS詠唱判斷是哪一個。

![超暗影](https://img.game8.jp/10299664/d94405672986aa470ffb96024a33bbd3.png/show)

①詠唱超暗影時，分身攻擊同時會有直線分攤著彈，對象者以外的7人一起分攤。

![碎裂尖甲](https://img.game8.jp/10299690/ed584bc11d08328e694ed68375ea8640.png/show)

②詠唱碎裂尖甲時，坦克1人DPS1人治療1人為對象的扇形範圍攻擊著彈，按職業集合處理。

![超暗影中央](https://img.game8.jp/10299689/44dd4fdfd479dd6306c6211130dfda17.png/show)

超暗影時，在中央側處理分身攻擊也同樣7人分攤。

![碎裂尖甲中央](https://img.game8.jp/10299674/d5aef26048b0bb33584d9bf311680f85.png/show)

在中央側處理分身攻擊時的碎裂尖甲，安全區變窄要注意。

#### 超暗影是7人集合分攤

超暗影是單純的直線分攤攻擊，分身攻擊對象以外的7人集合分攤即可。

#### 碎裂尖甲按職業集合

碎裂尖甲是以坦克1人、治療1人、DPS1人為對象的扇形範圍攻擊。面向BOSS左側坦克2人、中間DPS4人、右側治療2人，按職業集合處理。

### 傾盆大貓

採用犬丸式的處理法。傾盆大貓是接線者的扇形範圍攻擊，加上距離BOSS最近和最遠的人的分攤攻擊，連續4次。坦克2人各取1條線，在北側BOSS目標圈外的位置站好處理扇形攻擊。D1D2H1在BOSS南側目標圈內、D3D4H2在BOSS南側外圍分別集合承接分攤。坦克在第1次扇形後開無敵，用無敵承接第2、3次。第4次前無敵結束，將線交給近戰DPS並交換位置處理第4次。

![傾盆大貓①](https://img.game8.jp/10299711/a01b006875c6671adb98b8d5f525eb5a.png/show)

①BOSS的2條線由坦克接取，MT在西北、ST在東北處理扇形範圍。H1D1D2在BOSS南側目標圈內、H2D3D4在南側外圍集合。

![傾盆大貓②](https://img.game8.jp/10299714/45b11dcaa7ca09e7eaf7a138d4c4ed87.png/show)

②接線坦克承接扇形範圍攻擊，其他人承接3人分攤x4連續。※坦克第2、3次用無敵承接。

![傾盆大貓③](https://img.game8.jp/10299724/2c4bbc8d50399d92b73794f14434488e.png/show)

④第3次結束後，MT和D1、ST和D2交換位置承接第4次。機制結束。

## 重點提示

:::tip 分身記憶系統
BOSS使用「靈魂之影」來讓分身學習攻擊模式。注意觀察分身會重現哪個方向的攻擊。
:::

:::caution 模仿之貓判斷
- 右手亮 + 蹲下 = **擊飛**（會移動）
- 左手亮 + 站立 = **暈眩**（不移動）
- 善用單格角落解法可大幅簡化處理
:::
