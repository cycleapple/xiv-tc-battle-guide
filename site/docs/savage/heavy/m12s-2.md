### 境中幻夢（ワンエース式）

:::caution 超長機制
境中幻夢是持續3分鐘以上的超大型機制。攻擊會先**預覽**再**延遲執行**，且最先記憶的攻擊最後才發動。需要記住大量資訊。
:::

機制分為4個階段（P1\~P4），預覽和執行穿插進行：

**流程概覽：**

| 步驟 | 內容 |
|------|------|
| 1 | P4-1預覽：模仿細胞（記住X字或十字哪個先）。交換1↔B和4↔C位置，按巨集 |
| 2 | 心象投影1：場地變為夢境 |
| 3 | P1預覽：複製1（3個分身預覽，記住12安全或34安全） |
| 4 | 心象投影2：回到現實 |
| 5 | P2預覽：複製2（記住A是分攤還是圓形） |
| 6 | P1執行：心象投影3，P1分身攻擊發動 |
| 7 | 飛龍隕石：高傷全體，B/D島形成。MT組去D、ST組去B |
| 8 | P3預覽：墮落+登天奧秘（4座塔/島，DPS或TH獲得光耐性降低。確認踩塔分配） |
| 9 | P2執行：心象投影4，分攤+圓形發動（2組） |
| 10 | P3執行：心象投影5，踩塔+近/遠發動 |
| 11 | P4-2預覽：空間裂斷（3個分身重現，確認南北哪個被吸入裂縫） |
| 12 | P4-1執行：心象投影6+重現，模仿細胞攻擊發動（X字或十字分攤） |
| 13 | P4-2執行：心象投影7，島上分身攻擊發動 |
| 14 | P4-1+2執行：心象投影8，剩餘模仿細胞+被吸收分身扇形攻擊 |

#### P3踩塔類型

| 塔類型 | 效果 | 減益 |
|--------|------|------|
| **光闇（紫色）** | 站在塔外緣將光線朝外。附加近+死亡宣告（需驅散） | 往南側數字標記角落 |

### 雙旋踢 → 境中奇獄×2 → 狂暴

境中幻夢結束後：雙旋踢（坦克強攻）→2次境中奇獄（全體攻擊）→狂暴詠唱。

:::caution 假動作注意
每次境中奇獄前，BOSS會使用空的複製動作（無傷害）。**不要在假動作時使用減傷**，留給真正的境中奇獄。第2次境中奇獄傷害高於第1次。
:::

## 重點提示

:::tip 複製屬性判斷
第1波後獲得火/暗耐性降低。第2波必須接受**相反屬性**的攻擊。無減益者視同火減益處理。
:::

:::tip 變異細胞α/β規則
α＝不能受傷（坦克要換仇恨）、β＝必須受傷。細胞變異後永遠互換。陰怒：遠時β到外圈α到BOSS下，近時相反。
:::

:::caution 境中幻夢記憶重點
1. P4-1：X字還是十字先？
2. P1：12安全還是34安全？
3. P2：A是分攤還是圓形？
4. P3：塔的類型和光耐性降低對象
5. P4-2：南北哪個分身被吸入？

**最先記住的最後才用**——P4-1的X字/十字是最後才執行的。
:::

:::caution 空間裂斷速記
安全側被吸收＝目標圈外（遠離BOSS）。對面被吸收＝目標圈內（靠近BOSS）。
:::
