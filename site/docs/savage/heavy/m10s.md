## 重點提示

:::tip 球判定記憶法
破勢乘浪的球判定：**球2個＝分攤、球8個＝散開**。球少人多→集合分攤，球多人散→各自散開。
:::

:::tip 水牢處理要點
- 炎線碰水牢＝削血（好的），水線碰水牢＝減傷buff（壞的）
- H1H2進水牢，第1次炎線不交叉
- 可保留2分鐘爆發到水牢破壞後使用
:::

:::caution 狂浪騰空互換規則
第2次蛇奪浪需要根據攻擊類型互換：強攻＝坦克換、分攤＝治療換、散開＝近戰/遠程換。互換後留在該側不回原位。
:::

:::caution 分斷階段注意
- MT組固定西、ST組固定東
- 2人分攤時不要堵塞中央線
- 擊退前建議使用防擊退技能
:::
