## 機制攻略

### 天頂的主宰

全體物理傷害攻擊。本場戰鬥BOSS大部分攻擊為物理屬性，近戰請積極使用牽制。

---

H1D3 / H2D4（上方兩區）、MTD1 / STD2（下方兩區，★左右）

## 重點提示

:::tip 武器機制記憶法
- **鐮**（甜甜圈）＝靠近BOSS + 8方向散開
- **斧**（圓形）＝遠離BOSS + 中央集合分攤
- **劍**（十字）＝斜角站位 + 兩列4:4分攤
:::

:::tip 流星雨放置順序
治療→遠程DPS→近戰DPS，按巨集指定位置放置。坦克負責接外周隕石連線（MT:2/3標記、ST:4標記）。
:::

:::caution 王者隕石（ジジ式）判斷
- 連線來自對面島→留在本島踩塔
- 連線來自本島→跳到對面島踩塔
- 無連線→留在本島踩塔後接標記放範圍
:::

:::caution 碎心踢減傷分配
第1次全員分攤、第2次MT→ST無敵接力（單人無敵無法撐全程）、第3次雙坦全開減傷。騎士不要在火焰流使用神聖領域，留給碎心踢。
:::
