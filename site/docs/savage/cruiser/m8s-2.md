## 機制攻略

### 爆震（クエイガ）

![爆震](https://img.game8.jp/11265460/a307235952d69c99813edca1ec679bb9.jpeg/show)

以2名治療為對象的魔法屬性分攤攻擊。**島全體都是攻擊範圍**，因此治療必須分別在不同的島（從基準島往東西各一座島）處理。之後**爆震基本在西島（MT組）和東島（ST組）**處理。

:::caution 注意事項
承受爆震時盡量靠前方（靠近BOSS側），以防萬一脫離對面治療的減傷或回復範圍。此外，**4人未滿的分攤傷害會暴增**，若人數不足，不如1人承受以將犧牲降到最低。
:::

### 魔光

![魔光①](https://img.game8.jp/11265458/300d6b4b2d2bb1828ece2c1b197f09df.jpeg/show)

①魔光詠唱前按此方式散開。由北往南依序為 坦克 > 遠程D > 近戰D > 治療。

![魔光②](https://img.game8.jp/11265457/9d03aa47f1df0759e5d4f41056d0e878.jpeg/show)

②外周出現浮游砲，玩家身上附加魔光標記。

![魔光③](https://img.game8.jp/11265459/e6e7966704451115282b868573024787.jpeg/show)

③各島上保持只有1個魔光標記的方式移動。

![魔光④](https://img.game8.jp/11265456/c100659713f2d80b2036a1314e99d2fb.jpeg/show)

④浮游砲的攻擊範圍與魔光攻擊同時發動，機制結束。

5名玩家被附加藍色標記，各島背後的浮游砲會施放半面範圍攻擊。標記對象所在的島會受到魔法屬性攻擊，因此**標記對象必須各島散開**。標記攻擊**同時各島背後的浮游砲也會施放半面範圍攻擊**，確認浮游砲位置後迴避半面範圍。

#### 魔光的移動判斷

![魔光判斷](https://img.game8.jp/11265508/5c5fb0e73b8b71be58922e7d52a2da2d.jpeg/show)

| 條件 | 行動 |
|------|------|
| 魔光3個 | 北側的人去北島、中間的人原地不動、南側的人去南島（基準島） |
| 魔光2個 | 北側的人原地不動、南側的人去南島（基準島） |

MT組、ST組在東西島散開，由北往南按 坦克 > 遠程D > 近戰D > 治療 的順序排列。根據該島上魔光標記的數量判斷是否移動。

### 風震魔印（風震の魔印）

#### 正規處理（坦克2人處理範圍與減益）

![魔印①](https://img.game8.jp/11265595/127b46faef3c1d20673534a429dc318c.jpeg/show)

①剛刃一閃後各玩家按此方式在島上就位。

![魔印②](https://img.game8.jp/11265596/5975144d57cdb3793e1923a1df37462c.jpeg/show)

②風震魔印詠唱時對減益對象附加標記。**坦克以外移動到被標記坦克的隔壁島**。未被標記的坦克挑釁取得仇恨1位。坦克之間稍微靠近以便傳遞減仇。

![魔印③](https://img.game8.jp/11265597/83807ad4ccaad1cdaaa4bb179e7d8746.jpeg/show)

③魔印發動。被標記坦克施放巨大圓形範圍（不要碰到DPS治療），被標記DPS施放分攤攻擊，仇恨1位受到扇形範圍 + 束縛。

![魔印④](https://img.game8.jp/11265675/090d4d9f8d06a2f1ca15752d3faf0598.jpeg/show)

機制結束。之後風狼陣開始，各玩家就位。

坦克1人和DPS治療6人中的1人會被附加標記。坦克以外的標記減益是**分攤攻擊，由坦克以外的6人處理**。

被標記的坦克會在時間經過後發動**巨大圓形範圍攻擊**，留在原本的島上等待處理。同時**無詠唱地對仇恨1位施放束縛 + 巨大扇形範圍**，因此DPS + 治療側的標記對象要移動到被標記坦克的隔壁島處理減益。

:::tip 簡記
DPS和治療去被標記坦克的隔壁島，非常痛要確實使用減傷。
:::

### 風狼陣

![風狼陣](https://img.game8.jp/11265669/49096fd3c624ec99cdc7cee310cc26c6.jpeg/show)

4座島上各出現1座2人塔，按指定位置散開分別處理各塔。塔處理結束後，**DPS立即移動到坦克和治療處理塔的島上**，為下一個機制做準備。

### 魔狼戰形·颶風之相 + 雙牙暴風擊

魔狼戰形·颶風之相的詠唱會將傳送門再使用禁止減益延長至7秒，之後處理雙牙暴風擊機制。雙牙暴風擊詠唱開始後**隨機對象被附加線**。線是2人分攤 + 著彈點逐漸擴大的圓形範圍，**必須在島的外側處理**。線著彈同時BOSS會對**距離近的人施放2人分攤攻擊**，因此要邊處理線邊處理分攤攻擊，共4次。

線的處理採接力形式：最初TH組接線，經由DPS組傳遞到對面的TH。期間同時靠近處理2人分攤攻擊。

<video controls width="100%">
  <source src="https://img.game8.jp/11265663/c966025b3793cb48fa8534f3a6407b0d.mp4/show" type="video/mp4" />

| 順時針範例 | 逆時針範例 |
|------|------|
| ![順時針](https://img.game8.jp/11268150/83e8bef279c78e6578a50ba09bc3742f.jpeg/show) | ![逆時針](https://img.game8.jp/11268149/66f6057a5a9fece6f4890a0bdb9e3bbf.jpeg/show) |

光牙召喚的詠唱會在各島背後設置巨大浮游砲，接著迴天動地連續5次施放5種範圍攻擊。**迴天動地的範圍攻擊同時背後的浮游砲也會施放半面攻擊**，要找到兩邊都能迴避的位置。迴天動地下一次的範圍可以根據詠唱中出現的旋轉方向和隔壁島的範圍攻擊來判斷。

AOE預兆的排列順序有規律，但初始AOE的位置和順逆時針是隨機的。後方安全的下一次必定是BOSS附近安全，順時針時左右安全後接甜甜圈，逆時針時甜甜圈後接左右安全。浮游砲的半面範圍也別忘了確認。

:::tip 開怪時機
如果在爆震第2次開爆發，第2次的2分爆發大約在迴天動地結束時回來，做好準備。
:::

#### 迴天動地後接爆震→魔光→雙牙擊

迴天動地後會接序盤出現過的爆震→魔光→雙牙擊的一連串流程，MT組和ST組要迅速移動到東西的島上。

### 魔狼戰形·咒刃之相 + 獨狼的詛咒 + 風狼陣 + 鎧袖一觸（第2次）

#### Alice式解法

![孤狼①](https://img.game8.jp/11268157/9e58dbcf9363ccfa4d35a95dfd7e993d.jpeg/show)

①按上述定位就位。

![孤狼②](https://img.game8.jp/11268158/e1137b8e030084cc19111c8dd800f912.jpeg/show)

②TH和D各自被附加線。

![孤狼③](https://img.game8.jp/11268159/2189694fd9d772e408b2a196894a30e9.jpeg/show)

③確認藍線（遠離）或綠線（靠近），按指示移動（詳見下方）。

![孤狼④](https://img.game8.jp/11268162/c45ccaed26ceb28a708b81364b16c9f6.jpeg/show)

④塔出現。有隨機性的只有北側2座塔。北東的坦克和DPS根據塔需要的人數決定是否移動。

![孤狼⑤α](https://img.game8.jp/11268163/a46754787a8003104436858870169a50.jpeg/show)

⑤α 上圖情況北西為1人塔，藍線DPS移動去踩塔。

![孤狼⑤β](https://img.game8.jp/11268164/23dfbff40c0305da1dea8416ee1630ed.jpeg/show)

⑤β 上圖情況北西為2人塔，綠線的坦克和DPS移動去踩塔。

![孤狼⑥](https://img.game8.jp/11268167/411b1afea6520b7f670cba54595adf7d.jpeg/show)

⑥立即接鎧袖一觸，找到左右1次傳送就能迴避的位置。之後馬上接魔光，這裡很容易出事故。

魔狼戰形·咒刃之相的詠唱會將傳送門再使用禁止減益延長至7秒，之後處理獨狼的詛咒機制。**獨狼的詛咒詠唱前各職責要在各自的初始位置等待。**詠唱後全員被附加遠離線（藍線）或靠近線（綠線），之後在維持適當線距離的同時處理風狼陣的塔機制。

**塔的出現模式只有2種**，北側的坦克和DPS交換或不交換位置，只有2種調整模式。

#### 各職責行動

| 職責 | 行動 |
|------|------|
| **坦克** | 移動到北西島等待。藍線（遠離）：與D相連，移動到西島。風狼陣在西島處理（不動）。綠線（靠近）：與D相連，移動到北東島。該島有2塔則不動，沒有則移動到北西島處理2塔。 |
| **DPS** | 移動到東島等待。藍線（遠離）：往北伸（與T相連）→原地不動；往南伸（與H相連）→移動到北東島。風狼陣：與坦克相連者在東島處理；與治療相連者，該島有1塔則不動，沒有則移動到北西島處理1塔。綠線（靠近）：往北伸（與T相連）→移動到北東島；往南伸（與H相連）→移動到南方基準島。風狼陣：與坦克相連者，該島有2塔則不動，沒有則移動到北西島處理2塔；與治療相連者在基準島處理3塔。 |
| **治療** | 留在基準島不動。坦克會被打得很痛，盡可能提前做好補血準備。風狼陣：基準島會出現3座塔，踩進去即可。 |

#### 鎧袖一觸第2次後的魔光散開

![魔光散開](https://img.game8.jp/11268198/8b32e64110702b906d329de7e2657008.jpeg/show)

上圖為戰鬥前半的魔光整列，但實際上很難排得這麼整齊，要注意觀察隊友位置。

鎧袖一觸第2次後傳送裝置會恢復為白色，再使用減益的秒數變回較短的狀態。鎧袖一觸結束後**看到傳送裝置變白就立刻移動回爆震處理用的島上，用之前相同的判斷方法處理魔光**。

:::caution 魔光散開注意
要像最初的魔光那樣全員整齊排列非常困難，但MT組和ST組分東西兩側是必須的。確認自己負責側（MT組或ST組）的魔光是3個還是2個。
:::

### 八連光彈 + 剛刃一閃 ~ 剛刃一閃·終

#### 81128處理法

<video controls width="100%">
  <source src="https://img.game8.jp/11265692/14ca72936a09c277b09dcecb5a516da3.mp4/show" type="video/mp4" />
</video>

八連光彈對島施放分攤攻擊，剛刃一閃逐一破壞島嶼。每次剛刃一閃破壞島嶼時BOSS會獲得增益，**八連光彈的分攤攻擊會越來越痛**，要確實商量減傷分配。**八連光彈的分攤可以用坦克無敵或坦克2人全減傷承受**。八連光彈 + 剛刃一閃共進行5次，全部島嶼被破壞後即為時間切斷。

81128處理法是指五次八連光彈中：「**第1次8人承受→第2次坦克無敵（1人）→第3次坦克無敵（1人）→第4次坦克2人全減傷承受→第5次8人承受**」的方法。

第1次全員8人承受，攻擊結束後在剛刃一閃詠唱時移動到隔壁島。第2～4次坦克以外在八連光彈詠唱中就移動到隔壁島。

## 重點提示

:::tip 前半溫存LB可在後半打2次LB3
若前半能溫存LB，後半開場和終盤各打一次LB3可大幅減輕DPS壓力。即使只打1次也能過，但2次會更輕鬆。
:::

:::tip 爆發時機
後半開始時1分技能已回來但2分技能還沒完全回來。開場先放1分技能，等到爆震第2次左右再開2分爆發。之後按回轉打即可。後半開場使用藥品，終盤八連光彈附近藥品CD會再次回來，可打2次藥。
:::

:::caution 風震魔印
DPS和治療記住「去被標記坦克的隔壁島」即可。傷害非常高，確實使用減傷。
:::

:::warning 獨狼的詛咒後的魔光
鎧袖一觸第2次結束後馬上接魔光，是最容易出事故的地方。傳送裝置變白後立刻移動，確認自己負責側的魔光數量並正確判斷。
:::
