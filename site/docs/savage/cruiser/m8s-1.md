## 機制攻略

### 1. 空間斬（空間斬り）

物理屬性全體攻擊。傷害約18萬非常痛，請務必使用牽制等減傷來壓低傷害。全體攻擊後會立刻接2次約8萬傷害的普攻，減傷能覆蓋到普攻的話更好。

| **風＋十字** | **風＋X字** |
| ![風十字](https://img.game8.jp/11256742/3e4ee5d62f167698ee79238e3e757eee.png/show) | ![風X字](https://img.game8.jp/11256741/91d525322ad59ec0d9fa093cf52fe399.png/show) |

BOSS詠唱土或風之魔技。**土**的情況為BOSS中心圓形範圍＋8人各自的扇形範圍攻擊；**風**的情況為甜甜圈範圍＋TH4人或DPS4人的配對分攤扇形範圍攻擊。此外，射出的浮游砲會釋放十字或X字範圍。

土/風與浮游砲的攻擊會複合出現，根據各自的安全區來應對。全體範圍傷害約18萬非常痛，請使用減傷壓低傷害。

### 3. 掃擊/旋擊群狼劍

#### 旋擊群狼劍

| | |
|:---:|:---:|
| ![廻1](https://img.game8.jp/11257848/add5f57120245dc3e9e44f2edc5716bd.png/show) | ![廻2](https://img.game8.jp/11257847/e24bfff0a9df9fca5ab6a7c12d95a02b.png/show) |

旋擊群狼劍是BOSS為中心的圓形範圍，需要遠離BOSS。第2擊會對仇恨1~2位施放扇形範圍、對治療施放分攤扇形範圍，按照上圖位置處理。

#### 掃擊群狼劍

| | |
|:---:|:---:|
| ![薙1](https://img.game8.jp/11257849/1f0d94f27df5a9d0b3e9b01cea6786a0.png/show) | ![薙2](https://img.game8.jp/11257850/ce5dce1427e50aed202a8d8c842426ca.png/show) |

掃擊群狼劍是BOSS發出巨大扇形範圍，需要靠近BOSS。第2擊同樣對仇恨1~2位施放扇形範圍、對治療施放分攤扇形範圍。

根據掃擊或旋擊的詠唱決定最後的安全區。會出現3個分身，分身的目的地和本體都會產生圓形範圍。本體移動到外周後，伴隨直線攻擊移動到對面。

移動後詠唱「掃擊」則BOSS面向中央施放扇形範圍，「旋擊」則BOSS中心圓形範圍攻擊。之後仇恨1位2位承接較粗的扇形範圍，治療承接分攤扇形範圍，在指定位置處理。

### 5. 連震擊＋空間斬（連震撃＋空間斬り）

10萬傷害的8次連續分攤攻擊。而且第3、5、8次還會伴隨普攻。非常痛，請務必使用減傷壓低傷害。此外之後還會接空間斬，要確實做好回復、減傷和護盾。

:::tip 牽制時機
近戰DPS在連震擊第1發發動時使用牽制，可以覆蓋到空間斬，非常推薦。
:::

---

### 6. 一刀兩斷

坦克2人承接的分攤直線強攻擊。2人承接也非常痛，分攤時要厚疊減傷。**無敵技有效，沒有特殊理由的情況下建議用坦克無敵技處理**。

### 8. 光狼召喚（小怪階段）

| | |
|:---:|:---:|

②屬性符號出現。坦克追著自身持有狼頭的反屬性符號，站在反屬性符號側的位置。DPS和治療站在坦克的對側。

| | |
|:---:|:---:|
| ![小怪3](https://img.game8.jp/11257609/6ab6c2b32949c9b0dc89b916c745f113.png/show) | ![小怪4](https://img.game8.jp/11257605/8784ea66e20a1ffe5fa31fa2086ab2fb.png/show) |

③標記出現，光狼陣發動。坦克和標記對象會被施放直線攻擊。注意不要將範圍朝向對面的狼頭。

④被魔法增傷減益消失後，風（早）減益的人立即去觸碰風屬性符號。

| | |
|:---:|:---:|
| ![小怪5](https://img.game8.jp/11257608/7b6261d23660a682ca644b169d075125.png/show) | ![小怪6](https://img.game8.jp/11257607/ef624eccf152f892d477dd0a8591c114.png/show) |

⑤觸碰符號後會附加4秒被魔法增傷減益。**土（早）減益的人要等到被魔法減益解除後才能觸碰符號**。

⑥被魔法減益解除後，土（早）減益的人立即去觸碰土屬性符號。③~⑥的流程重複3次，全員減益解除後擊殺小怪。

BOSS進入不可選中狀態，東西出現「風狼首」和「土狼首」。風塵咆哮發動後出現風/土符號，並給玩家附加減益（觸碰符號可解除），全員在解除減益的同時擊殺小怪。

#### 附加的減益

| 減益 | 說明 |
|------|------|
| 風/土之戒 | 戒與反屬性的狼頭才能造成傷害。觸碰符號解除咒痕減益後戒的屬性會反轉。擊殺1隻狼頭後全員的戒減益解除。 |
| 風/土之咒痕 | 觸碰同屬性符號可解除。觸碰順序為**風→土**。觸碰符號後會產生爆炸，對狼頭造成20%傷害（反屬性的情況），對玩家造成約13000傷害並附加**4秒被魔法傷害增加減益**，因此後觸碰的土減益玩家要等被魔法增傷減益消失後再觸碰。減益秒數為**「53 / 36 / 20」3種**，秒數短的先觸碰。風的人到了自己的回合要迅速觸碰符號，否則土觸碰時的被魔法增傷減益與光狼陣的魔法攻擊重疊會導致滅團。 |

#### 小怪階段時間軸

| 階段 | 說明 |
|------|------|
| ①風塵咆哮發動 | 玩家附加減益，小怪可攻擊 |
| ②光狼陣第1次詠唱 | 詠唱完成後，風咒痕（早）觸碰風符號→確認被魔法減益解除後，土咒痕（早）觸碰土符號 |
| ③光狼陣第2次詠唱 | 詠唱完成後，風咒痕（中）觸碰風符號→確認被魔法減益解除後，土咒痕（中）觸碰土符號 |
| ④光狼陣第3次詠唱 | 詠唱完成後，風咒痕（遲）觸碰風符號→確認被魔法減益解除後，土咒痕（遲）觸碰土符號 |
| ⑤擊殺小怪 | |

小怪階段中解除減益的時機以小怪詠唱的「光狼陣」為基準。例如「我是遲減益，所以光狼陣第3次詠唱後觸碰符號」這樣記住即可。

:::tip 忘記減益秒數時
如果忘記自己的減益秒數，在「光狼陣」詠唱中確認減益秒數低於20秒就是自己的回合。換句話說，不用記住遲早中的順序，只要光狼陣詠唱中減益低於20秒就輪到自己即可。
:::

## 重點提示

:::tip 前半三大難關
「千年風化」「大地之怒」「幻狼召喚」是前半特別容易出事故的3個機制，請大量練習直到能完美處理。前半穩定了，進入後半的機率就會大幅提升，是通關的捷徑。
:::

:::tip LB使用建議
- **前半突破～後半練習目的**：建議在前半積極使用LB3。使用LB3突破前半的基準大約是小怪階段前BOSS HP 47~48%左右。
- **通關目的**：也可以前半只用LB1，後半用2次LB3（LB133策略）。不過前半只用LB1的話DPS檢定會更嚴格，小怪前BOSS HP需要在45%左右。
:::
